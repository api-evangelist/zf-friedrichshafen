# ZF Friedrichshafen (zf-friedrichshafen)

ZF Friedrichshafen AG is a German global Tier 1 automotive supplier headquartered in Friedrichshafen. ZF designs and manufactures driveline, chassis, active and passive safety, and ADAS technology for passenger cars and commercial vehicles. ZF acquired TRW Automotive in 2015.
